# Real-Time On-Screen Drawing

**Objective:** Develop a real-time interactive system that enables users to draw directly on the screen using object movement, with the ability to reset and restart the drawing process at any time;

**Methodology:** Implemented object detection and movement tracking to capture continuous motion, updating drawing strokes in real time; 

**Result:** Successfully created an interactive real-time application that allows users to draw on a virtual whiteboard that does not physically exist but is rendered live on the video feed;
